GitHub Pages 로 이동

1. Step 1: Add homepage to package.json
  "homepage": "https://myusername.github.io/my-app" 를 복사해서 package.json 끝에 붙여넣기
  "homepage": "https://sungseurun.github.io/kurly_cra_typescript_sause/" 로 수정하기 (내 주소)

2. Step 2: Install gh-pages and add deploy to scripts in package.json
  npm install --save gh-pages 복사해서 터미널에 설치하고
  package.json 에서 "gh-pages": "^4.0.0", 이거 되어있나 확인하기

3.
  "predeploy": "npm run build",  복사하고
  "deploy": "gh-pages -b master -d build", 복사해서
  package.json 에 있는 Debug에 붙여넣기

4. Step 3: Deploy the site by running npm run deploy
  npm run deploy

5. 배포 끝 완료.

  깃허브 배포 위치에서 1~3분 정도 기다린다.

6. 웹브라우저에서 띄운다.
